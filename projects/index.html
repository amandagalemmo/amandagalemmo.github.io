Minesweeper is a Roguelike

Born from the idea that Minesweeper fits the definition of a roguelike as laid out in Wikipedia, I created a simple dungeon-crawler-esque verison of Minesweeper where you are just some little guy in a procedurally-generated world. Currently in progress, with some visual bugs that need to be ironed out + some feature additions planned for the future. Play it here!
